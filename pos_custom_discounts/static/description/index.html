POS Custom Discount

For Odoo POS we have come up with this module named POS Custom Discount which allows seller to give discount on single order line as well as on whole order.It has changed the old functionality of discount button. In this module ,when you click on discount button a new window will appear which contains those discounts that are configured from backend. It also allows seller to create custom discount(new discount) in running POS session .In every custom discount a reason will be mentioned that is saved in backend and will also reflect in orderline and receipt.

Salient Features
  • POS Custom Discount allows seller to provide discount on single orderline as well as on complete order
  • Seller can create multiple discounts on different basis from backend.
  • Only those discounts will be available in POS Session that are selected in current Point Of Sale Configuration.
  • Seller can create custom discount in running POS session after entering his security pin and reason for discount
  • Only authorized POS users are allowed to create custom discount
  • The Reason for custom discount will be mentioned on receipt and get saved in backend also.

Creating custom discount

Click on custon discount button to create a custom discount

After clicking on custom discount button, user authentication will be done. A valid POS user needs to enter Security Pin to create custom discount. This ensures that only the authorized POS user can create custom discount.

After password authentication,this window will be shown as shown below. It contains discount percentage and discount reason. Reason for custom discount will be mentioned on receipt, orderline and get saved in backend also.

If seller click on selected line button then discount will be applied on selected order line(selected product in cart) with reason.

If seller click on whole order button then discount will be applied on all product in cart(complete order) with reason as shown bellow.
